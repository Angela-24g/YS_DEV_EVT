# Ping 통신 테스트

## 1. 사용배경
통신 테스트를 통과한 차량 단말기 위치에 해당하는 링크의 평균 시간을 맵으로 표현하고 싶다.
(2) 실시간 차량 위치 및 링크 ID 세팅(각 위치에 따른 아이디 추출)

## 2. 사용기술
- springboot PVD kafka Listener
- Polygon
- SimpleFeature

## 3. 설계 / 구조
features를 직접 만들어서 넣음(기준Id, 구역명, 지역 코드, 지역 구, GEOM(4326 폴리곤 데이터)) -> 이후 feature를 
```bash
SimepleFeature 구성 (현재 예시)
 - 속성1(id)
 - 속성2(name)
 - 속성3(geom)
```
