# proxy 패턴 사용

## 1. 사용배경
프로젝트 2개를 연결하기 위해서 proxy(대리자) 코드가 있어야 함
이를 통해 클라이언트와 서버 간 연결을 할 때 중간 단계에서 처리를 할 수 있도록 해야했음

## 2. 사용기술
- springboot restTemplate
- restController
- UriComponents

## 3. 설계 / 구조
uriComponents를 이용하여 UriComponentsBuilder로 빌드를 하고, 이렇게 하여 uri를 생성.
그 후, restTemplate을 이용하여 처리하고자 하는 서버에 요청을 보내는 방법
