# Ping 통신 테스트

## 1. 사용배경
통신 테스트를 통과한 차량 단말기 위치에 해당하는 링크의 평균 시간을 맵으로 표현하고 싶다.
(3) 추출된 링크 ID를 매칭하기(ol 함수 관련)

## 2. 사용기술
- GeometryHelper
- geometryFactory, Envelope

## 3. 설계 / 구조
ping을 이용하여 통신 테스트를 하고, 데이터를 읽고자 함.
그 후, restTemplate을 이용하여 처리하고자 하는 서버에 요청을 보내는 방법

## 4. 코드
```java
    // 1. 읽은 위경도 값으로 GEOMETRY의 위치를 얻어온다.
     Coordinate coord = new Coordinate(lon, lat);
    Point point = geometryFactory.createPoint(coord);

    // Transform 5181(4326 -> 5181)
    point = (Point) GeometryHelper.transform5181(point);
    coord = point.getCoordinate();

    Envelope searchEnv = new Envelope(coord);

    // Broad phase: query index
    List<?> candidates = index.query(searchEnv);

    // Narrow phase: exact check
    for (Object candidate : candidates) {
      BuffLinkResult link = (BuffLinkResult) candidate;
      if (link.getGeom().contains(point)) {
        resultIds.add(link.getLinkId());
      }
    }
    
    
    // 2. Geometry.transform5181
    return transformGeometry(geometry, "EPSG:4326", "EPSG:5181");
```
